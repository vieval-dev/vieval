# Introduction

Vieval is a Vitest-based evaluation framework for agents, models, and model-backed workflows.

This documentation site is intentionally minimal for now. It gives the project a VitePress structure, navigation, and theme setup so the guide can grow without changing the docs foundation again.

## What belongs here

- Concepts and terminology for evaluation flows.
- Step-by-step usage guides.
- Examples for tasks, assertions, model executors, and reporters.
- Links to reference pages generated or maintained from package APIs.
