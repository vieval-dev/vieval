# Vieval 指南

Vieval 用于在产品代码旁定义可重复执行的评测，按模型和场景展开任务，并同时保留便于人阅读的结果与可供工具处理的证据。

## 从这里开始

- [为什么选择 Vieval](/zh-hans/guide/why)：了解 Vieval 解决的问题及其能力边界。
- [快速开始](/zh-hans/guide/getting-started)：运行一次确定性的评测，不调用模型提供方，不需要凭据，也不会产生模型调用费用。
- [核心概念](/zh-hans/guide/core-concepts)：了解本指南使用的对象与执行生命周期。
